## ✨ 核心亮点

* 🎨 **神仙颜值**：精心设计的古风与现代融合场景，搭配超高颜值的许愿道具。
* 🍃 **随风飘动**：独创风力仿真算法，所有挂在树上的道具都会**随机地、轻柔地随风飘动**。
* ✍️ **自由书写**：全中文支持，心愿文字内容**完全无限制**，自由抒发真实情感。
